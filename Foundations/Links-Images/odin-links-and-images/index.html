This is an example of an anchor element.

click me

Clicking does nothing as it needs a destination to which it will link. That destination is the URL https://www.theodinproject.com/about. The actual command then becomes:

<a href="https://www.theodinproject.com/about">click me</a>

click me

So the <a> tags nail down an anchor and the href is set to the destination of the anchor to make it a link.

About

The href may be a fully qualified domain name, or may resemble a relative file pathname just like a unix filepath name. Think of fully qualified domain name as the root of an absolute file pathname.
